# 📢 Telegram уведомления для публикации в соцсетях

## Обзор

Теперь при публикации новостей ВКонтакте и Одноклассники отправляются уведомления в Telegram.

---

## 🔔 Типы уведомлений

### ✅ Успешная публикация

```
✅ Публикация в ВКонтакте

Новость: Заголовок новости
ID: 123
Изображений: загружено 5 (ошибок: 2) ⚠️

🔗 Открыть новость
```

**Когда отправляется:**
- Публикация успешно завершена
- Показывает количество загруженных изображений
- Показывает количество ошибок (если есть)

### ❌ Ошибка публикации

```
❌ Публикация в Одноклассники

Новость ID: 123
Заголовок: Заголовок новости
Ошибка:
  API error: timeout
  
🔗 Редактировать новость
```

**Когда отправляется:**
- Новость не найдена в базе данных
- Ошибка API социальной сети
- Исключение в процессе публикации
- Все попытки исчерпаны

### 2. Создание бота (если нет)

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Введите имя бота
4. Скопируйте токен в `config/params.php`

### 3. Узнайте Chat ID

1. Добавьте бота в чат/канал
2. Отправьте любое сообщение
3. Откройте: `https://api.telegram.org/bot<ТОКЕН>/getUpdates`
4. Найдите `"chat":{"id": -1001234567890}`

## 📊 Что показывают уведомления

### Успешная публикация

| Поле | Описание |
|------|----------|
| **Новость** | Заголовок новости |
| **ID** | ID новости в базе данных |
| **Изображений** | Количество загруженных картинок |
| **ошибок** | Количество неудачных загрузок (если > 0) |
| **⚠️** | Индикатор проблем с загрузкой |

### Ошибка публикации

| Поле | Описание |
|------|----------|
| **Новость ID** | ID новости |
| **Заголовок** | Заголовок новости (если найдена) |
| **Ошибка** | Текст ошибки или исключения |
| **🔗** | Ссылка на редактирование новости |

## 🎛️ Логика работы

### Сценарий 1: Всё успешно

```
1. Джоб запускается
2. Новость найдена
3. Картинки загружаются (некоторые с ошибками)
4. Публикация успешна
5. → Отправляется уведомление ✅ с количеством ошибок
```

### Сценарий 2: Критическая ошибка

```
1. Джоб запускается
2. Новость не найдена
3. → Отправляется уведомление ❌ "Новость не найдена"
```

### Сценарий 3: Ошибка API

```
1. Джоб запускается
2. Картинки загружаются
3. Ошибка при публикации в API
4. → Отправляется уведомление ❌ с текстом ошибки
```

## 🔧 Методы в SocialPublishJob

### sendTelegramNotification()

```php
/**
 * Отправляет уведомление в Telegram
 *
 * @param string $message Текст сообщения
 * @param bool $isError Это ошибка (красное сообщение)
 */
protected function sendTelegramNotification(string $message, bool $isError = false): void
```

### sendSuccessNotification()

```php
/**
 * Отправляет уведомление об успешной публикации
 *
 * @param News $news
 * @param int $uploadedImagesCount Количество загруженных изображений
 * @param int $failedImagesCount Количество неудачных загрузок
 */
protected function sendSuccessNotification(News $news, int $uploadedImagesCount = 0, int $failedImagesCount = 0): void
```

### sendErrorNotification()

```php
/**
 * Отправляет уведомление об ошибке публикации
 *
 * @param News|null $news
 * @param string $errorMessage Текст ошибки
 */
protected function sendErrorNotification(?News $news, string $errorMessage): void
```

## 🆘 Решение проблем

### Уведомления не приходят

**Проверьте:**
1. Параметры в `config/params.php`
2. Бот добавлен в чат
3. У бота есть права на отправку сообщений

### Ошибка "Unauthorized"

**Решение:**
```php
// Проверьте токен
curl "https://api.telegram.org/bot<ВАШ_ТОКЕН>/getMe"
```

### Ошибка "Chat not found"

**Решение:**
1. Добавьте бота в чат
2. Отправьте сообщение в чат
3. Проверьте Chat ID через getUpdates
